# PML Website Next

Final stack website development for Pharma Metric Labs.

## Tech Stack

- Frontend: NextJS, TypeScript, Tailwind CSS
- Backend: NestJS, TypeScript
- Database: PostgreSQL
- ORM: Prisma
- Cache / Protection: Redis

## Project Structure

apps/web - NextJS frontend
apps/api - NestJS backend
packages - Shared packages or utilities

## Development

Frontend:

cd apps/web
npm run dev

Backend:

cd apps/api
npm run start:dev
